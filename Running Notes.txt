Adding Data Disk to Azure VM (Windows & Linux)
------------------------------------------------
A Data Disk is an additional disk attached to a VM for Storing Application Data, Databases, Logs, Backups, etc.


Example:

OS Disk =127 GB (Windows 127/Linux 127,30 GB OS installed)
Data Disk = 128 GB (Store files, databases, logs)

Steps: add data disk in azure portal
1. Navigate to your VM in Azure Portal.
2. Select Settings -> Disks.
3. Click + Create and Attach a new disk.
4. Enter:
	- Disk Name
	- Disk Size 
	- Disk Type (Premium/Standard SSD)
5. Click Save.

Note: The disk is now attached to the VM, but it must be initialized inside the OS.
